# 02 Model I/O

这个目录用于理解“模型输入输出”这件事：同一个大模型可以通过原生 SDK、LangChain 0.3、LangChain 1.x、厂商专用集成等方式调用，但输入结构、初始化方式和返回对象会有差异。

LangChain 集成库总览：<https://docs.langchain.com/oss/python/integrations/providers/overview>

| 文件名 | 核心想教会什么 |
| --- | --- |
| `01_ModelIO_Params.py` | 学会设置常见模型参数，重点观察 `temperature` 对输出随机性的影响，并认识 `content`、`response_metadata`、`usage_metadata`。 |
| `02_ModelIO_OpenAI.py` | 学会不用 LangChain、直接用 OpenAI SDK 调用 OpenAI 兼容接口，并理解原生返回结构为什么需要逐层取值。 |
| `03_ModelIO_0.3_ChatOpenAI.py` | 学会用 LangChain 0.3 的 `ChatOpenAI` 调用 OpenAI 兼容接口，并用多角色消息列表传入 system/user 内容。 |
| `04_ModelIO_1.0_Init_chat_model.py` | 学会用 LangChain 1.x 的 `init_chat_model` 初始化模型，并对比字符串输入和多角色消息输入。 |
| `ModelIO_DeepSeek.py` | 学会使用 DeepSeek 的 LangChain 专用集成 `ChatDeepSeek`，理解厂商独立包和 OpenAI 兼容接口路线的区别。 |
| `ModelIO_Qwen.py` | 学会使用通义千问的 LangChain 社区集成 `ChatTongyi`，并对比 `invoke` 一次性返回和 `stream` 流式返回。 |
