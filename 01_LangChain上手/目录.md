# 01 LangChain 上手

这个目录用于建立 LangChain 的第一层直觉：先能把模型调起来，再理解 0.3 和 1.0 写法差异，最后把示例代码整理成更像项目代码的形式。

LangChain 集成库总览：<https://docs.langchain.com/oss/python/integrations/providers/overview>

| 文件名 | 核心想教会什么 |
| --- | --- |
| `01_LangChain_v_0.3.py` | 学会 LangChain 0.3 常见写法：直接使用 `ChatOpenAI` 初始化 OpenAI 兼容模型，并理解 `invoke` 返回的是 `AIMessage`。 |
| `02_LangChain_v_1.0.py` | 学会 LangChain 1.x 的统一入口 `init_chat_model`，理解 `model`、`model_provider`、`api_key`、`base_url` 的作用。 |
| `03_LangChain_v_1.0.py` | 学会在同一个脚本里接入多个模型来源：阿里百炼 OpenAI 兼容接口和 DeepSeek 官方 provider。 |
| `04_LangChain_engineering.py` | 学会把演示代码工程化：用 `.env` 管理配置、封装初始化函数、使用日志、处理异常，并用 `if __name__ == "__main__"` 控制脚本入口。 |
